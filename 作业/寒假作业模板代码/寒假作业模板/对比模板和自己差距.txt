web项目规范
	首页应该以  index.css 命名
	存放图片 css 等不同关系的文件, 应该用不同文件夹分开. 例如 图片文件夹用image命名  css文件夹用css命名 

HTML
	1. HTML 标签是否运用规范, 合理
	2. 布局是否合理, 简洁.  
	3. class ID 命名 是否规范, 见命知意.   
	4. 代码排版是否整洁
	5. 使用定位是否合理  (浮动, 绝对定位, 相对定位, margin 等)

CSS 
	1. 代码排版是否整洁
	2. css 样式是否有累赘
	3. class 层级关系是否简洁.
